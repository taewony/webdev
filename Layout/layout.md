# 🍞 실전 예제로 배우는 웹 개발 기초 (Layout & CSS)

이 가이드는 `layout/index.html`과 `layout/css/style.css` 코드를 바탕으로 HTML/CSS의 핵심 개념을 설명합니다.

---

## 03장: 웹 페이지의 뼈대를 구성하는 HTML

### HTML 기본 구조 이해
모든 HTML 파일은 아래의 표준 구조를 따릅니다.
- `<!DOCTYPE html>`: 이 문서가 HTML5 문서임을 선언합니다.
- `<html>`: 전체 문서를 감싸는 루트 요소입니다.
- `<head>`: 브라우저 화면에는 보이지 않지만, 제목(`<title>`), 인코딩(`<meta>`), 외부 파일 연결(`<link>`) 등 중요한 정보를 담습니다.
- `<body>`: 사용자에게 실제로 보이는 모든 콘텐츠가 들어갑니다.

---

## 04장: 자주 쓰이는 기본 HTML 태그

### 4.1 제목, 이미지, 리스트 활용
- **제목 (`<h1>`~`<h2>`)**: 문서의 구조를 잡습니다. 예제에서는 브랜드명과 섹션 제목에 사용되었습니다.
- **이미지 (`<img>`, `<link rel="icon">`)**: 파비콘(아이콘) 설정이나 콘텐츠 이미지 삽입 시 사용합니다.
- **표 (`<table>`)**: 메뉴판처럼 격자 형태의 데이터를 표현할 때 유용합니다.

### 4.6 레이아웃 이해하기 (div, span, 시맨틱 태그)
- **`<div>`**: 특별한 의미는 없지만, 디자인을 위해 요소를 묶는 **상자** 역할을 합니다.
- **시맨틱 태그 (Semantic Tags)**: 브라우저와 개발자가 의미를 쉽게 파악할 수 있게 돕습니다.
  - `<header>`: 페이지 상단 (브랜드 슬로건, 로고)
  - `<main>`: 페이지의 핵심 본문
  - `<footer>`: 페이지 하단 (연락처, 저작권)
  - `<section>`: 본문 내의 논리적 구역 (메뉴 섹션 등)

## 05장: 웹 페이지를 풍성하게 만드는 CSS

### 5.1 스타일 적용 방법
예제에서는 **외부 스타일 시트 연결 방식**을 사용했습니다.
```html
<link rel="stylesheet" href="css/style.css">
```
이 방식은 HTML과 디자인(CSS)을 분리하여 유지보수가 매우 쉽습니다.

### 5.2 CSS 선택자 (Selector)
- **태그 선택자**: `body`, `header`, `h1` 등 특정 태그 전체에 스타일을 줄 때 사용합니다.
- **클래스 선택자 (`.`)**: `.container`, `.fancy-font` 등 특정 별명을 가진 요소에만 스타일을 줍니다. (가장 많이 사용됨)

## 06장: 꼭 알아야 할 CSS 기본 속성

### 6.1 폰트와 배경
- `font-family`: 글꼴을 설정합니다. 구글 폰트를 연결해 `fancy-font`를 만들었습니다.
- `background-image`: 배경 이미지를 넣습니다. `background-size: cover`를 쓰면 이미지가 잘리지 않고 상자를 가득 채웁니다.

### 6.4 박스 모델 이해하기 (Box Model)
모든 HTML 요소는 사각형 박스입니다.
1. **Content**: 실제 내용 (글자, 이미지)
2. **Padding**: 내용과 테두리 사이의 **안쪽 여백**
3. **Border**: 요소를 감싸는 **테두리**
4. **Margin**: 요소와 요소 사이의 **바깥쪽 여백**
- `box-sizing: border-box;`: 패딩을 넣어도 박스 전체 크기가 커지지 않게 고정해주는 필수 속성입니다.

### 6.5 기본 레이아웃 속성
- **display**: 요소를 어떻게 배치할지 결정합니다.
  - `flex`: 자식 요소들을 가로나 세로로 자유롭게 정렬할 때 씁니다. (헤더의 중앙 정렬에 사용됨)
- **position**: 요소를 원하는 위치에 고정시킵니다.
  - `relative`: 기본 위치를 기준으로 배치.
  - `absolute`: 부모를 기준으로 절대적인 위치에 배치 (푸터의 어두운 오버레이 처리에 사용됨).
- **text-align**: 텍스트 정렬 (left, center, right).
