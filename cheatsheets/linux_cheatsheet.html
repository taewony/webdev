Linux Cheat Sheet

리눅스 시스템 관리와 운영을 위한 핵심 명령어 모음입니다. 코드 블록을 클릭하여 복사하세요.

1 디렉터리 조작 (Directory)

현재 경로 확인 (pwd) pwd
디렉터리 목록 (ls) ls -al
디렉터리 이동 (cd) cd /path/to/directory
홈 디렉터리 이동 cd ~
디렉터리 생성 (mkdir) mkdir -p dir1/dir2
디렉터리 삭제 (rmdir) rmdir directory_name

2 파일 조작 & 보기 (File Operations)

파일 복사 (cp)cp source_file dest_file
디렉터리 복사cp -r source_dir dest_dir
파일 이동/이름변경 (mv)mv old_name new_name
파일 삭제 (rm)rm file_name
강제 삭제 (주의)rm -rf dir_name
빈 파일 생성/시간 변경touch file_name
전체 내용 보기 (cat)cat file_name
앞부분 보기 (head)head -n 10 file_name
끝부분 보기 (tail)tail -n 10 file_name
로그 실시간 보기tail -f /var/log/syslog
vi 에디터 실행vi file_name
화면 단위 보기 (more/less)less file_name

3 권한 & 소유권 (Permissions)

권한 변경 (chmod) chmod 755 file_name
실행 권한 추가 chmod +x script.sh
소유자 변경 (chown) chown user:group file_name
그룹 변경 (chgrp) chgrp group_name file_name

4 사용자 & 그룹 (User & Group)

사용자 생성 (useradd) useradd -m newuser
비밀번호 설정 (passwd) passwd newuser
사용자 정보 변경 (usermod) usermod -aG sudo newuser
사용자 삭제 (userdel) userdel -r username
그룹 생성 (groupadd) groupadd groupname
사용자 전환 (su) su - username

5 검색 (Search & Find)

내용 검색 (grep)grep "search_text" filename
재귀적 내용 검색grep -r "text" /path/to/dir
파일 찾기 (find)find / -name "filename.txt"
명령어 위치 (which)which java
매뉴얼 보기 (man)man ls

6 프로세스 관리 (Process)

프로세스 보기 (ps)ps -ef
실시간 모니터링 (top)top
PID로 종료 (kill)kill -9 PID
이름으로 종료 (killall)killall process_name
프로세스 검색 (pgrep)pgrep -fl process_name
프로세스 트리 (pstree)pstree

7 시스템 정보 (System Info)

OS 정보uname -a
CPU 정보lscpu
로그인 사용자whoami
접속자 목록w
블록 장치lsblk

8 리소스 & 디스크 (Resource & Disk)

메모리 사용량 (free)free -h
디스크 용량 (df)df -h
폴더 용량 확인 (du)du -sh directory_name
가상 메모리 통계vmstat 1
I/O 통계iostat

9 압축 & 아카이브 (Archive & Zip)

tar 압축tar -cvzf archive.tar.gz /path/to/dir
tar 해제tar -xvzf archive.tar.gz
zip 압축zip -r archive.zip folder
unzip 해제unzip archive.zip

10 패키지 관리 (dnf/rpm)

패키지 설치 (dnf)dnf install package_name
패키지 업데이트dnf update
패키지 검색dnf search keyword
설치된 패키지 확인 (rpm)rpm -qa | grep package_name

11 네트워크, 부팅, I/O

시스템 종료shutdown -h now
재부팅reboot
날짜 확인date
리다이렉션 (덮어쓰기)command > file.txt
파이프 (Pipe)ps -ef | grep process
마운트mount /dev/sdb1 /mnt/disk
