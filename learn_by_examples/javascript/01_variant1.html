변수

        var 변수명 = 값; 선언하면서 값을 초기화(할당), 어디에서든 사용가능, 함수내에서 지역변수, 호이스팅(끌어올려짐)
        let 변수명 = 값; 블록범위 지역변수 {code block}
        값을 지정후 변경가능

        const 변수명 = 값; const 상수, 변경 불가

변수 타입

        기본 타입(Primitive Type): 숫자, 문자열, 불리언, null, undefined, 심볼 -> 원시값
        참조 타입(Reference Type): 객체, 배열, 함수 -> 복합적인 값
        자료형 확인 typeof 연산자: F12 도구에서 확인 가능
